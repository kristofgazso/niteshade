InstallationÔÉÅ

niteshade requires Python 3.8 or higher.

niteshade binaries may be installed from PyPI using pip https://pypi.org/project/niteshade/.

$ pip install niteshade

Use of a virtual environment is strongly recommended.

Note

Installation with conda is not currently available. Anaconda users should install niteshade with pip inside an Anaconda environment.
